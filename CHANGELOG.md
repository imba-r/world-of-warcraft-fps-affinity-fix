# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-01-31
### Changed
- Converted console and log output to English

## [1.0.0] - 2026-01-31
### Added
- CPU affinity control (disables cores 0–3, enables 4–15 by default)
- RealTime priority with automatic High fallback
- Restart detection
- Timestamped logging
- Tested with World of Warcraft 12.0.1
